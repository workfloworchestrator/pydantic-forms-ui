# Tailwind SHADCN pydanticForms example

2 examples of pydantic forms one with plain tailwind and one with SHADCN and every pydantic forms optipn enabled.

## Add SHADCN component

Run from inside monorepo folder (`apps/example-tailwind`)

pnpm dlx shadcn@latest init
pnpm dlx shadcn@latest add button
pnpm dlx shadcn@latest add form
